## Installation

1. Install the global wallet SDK

```shellscript
npm install @blockscout/rootstock-global-wallet
```

2. Import  into your project

```shellscript
import "@blockscout/rootstock-global-wallet/ethereum";
```

The wallet is now available for usage!
