An explorer acts as the frontend to the blockchain backend. Not only are transactions, addresses, blocks etc highly available through the UI, direct access to data is available via the JSON RPC and REST APIs. 

- **Explorer Basics**
  - Searching transactions, addresses, blocks
  - Reading transaction details
  - Contract interactions
  - Token information
- **Advanced Features**
  - My Account setup and features
  - API keys for personal use
  - Watchlists and notifications
  - Mobile usage
- **Chain-Specific Guides**
  - Ethereum features (withdrawals, MEV, etc.)
  - L2-specific features
  - Cross-chain comparisons
