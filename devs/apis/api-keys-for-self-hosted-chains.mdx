---
title: "API keys for self hosted chains"
icon: "person-chalkboard"
---

Self-hosted chains are not a part of the [Blockscout PRO API infrastructure](/devs/pro-api), and API keys (when available) are allocated on a per-instance basis. 

If a chain supports MyAccount functionality -ie if you are able to login and connect to the instance with your wallet - then you can create a per-instance API key via the MyAccount panel.
