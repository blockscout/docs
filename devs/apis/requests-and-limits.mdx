|        **Type of API key**       | **rate limit<br/>value, req/sec** | **shared with<br/>other users?** | **API key is defined by <br/>env variable** | **rate limit for the API key managed by<br/>env variable** |                                                                                                        **Description**                                                                                                       |
|:--------------------------------:|:--------------------------------:|:-------------------------------:|:------------------------------------------:|:---------------------------------------------------------:|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------:|
| No API key                       | 5                                | Yes                             |                                            | API_RATE_LIMIT_UI_V2_WITH_TOKEN                           | The default rate limit for requests is 5 requests/second set via the `API_RATE_LIMIT_UI_V2_WITH_TOKEN` variable. This applies to global requests which do not originate from a whitelisted address or do not use an API key. |
| Individual API key               | 10                               | No                              |                                            |                                                           | An API key can be [created by the user in the My Account section](/using-blockscout/my-account/api-keys)                                                                                                                     |
| Rate limit, if IP is whitelisted | 25                               | No                              | API_RATE_LIMIT_WHITELISTED_IPS             | API_RATE_LIMIT_BY_WHITELISTED_IP                          | Rate limit of requests from whitelisted IPs                                                                                                                                                                                  |
| Per explorer instance API key    | 10                               | No                              | API_RATE_LIMIT_STATIC_API_KEY              | API_RATE_LIMIT_BY_KEY                                     | A per instance API key (*note this functionality was created prior to individual API keys and may have limited utility*)                                                                                                     |
| No rate limit API key            | Infinity                         | No                              | API_NO_RATE_LIMIT_API_KEY                  |                                                           | Admin API key                                                                                                                                                                                                                |

The default rate limit config of each API endpoint is described here https://github.com/blockscout/blockscout/blob/master/apps/block_scout_web/priv/rate_limit_config.json and can be re-defined via `API_RATE_LIMIT_CONFIG_URL` env variable.
